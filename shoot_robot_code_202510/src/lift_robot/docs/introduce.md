# 中国机器人大赛物流搬运赛项

## 业务流程

1. 运行到前方区域，观察AprilTag的ID，ID为1时去1号搬运，ID为2时去2号搬运；
2. 运行到摆渡点，向前运动到货架下方，抬起顶升机构，后退；
3. 运行到目标点，放下顶升机构，后退；
4. 返回出发区域。

## 顶升控制
高处：1:166 2:-205 3:114 4:-123
低处：1:136 2:-178 3:31 4:-37

## 示例代码描述

- apriltag_detect：二维码检测；
- lift_down：顶升机构下放；
- lift_put：顶升机构上升；
- movebase_client: 导航点测试；
- tf_echo: 待抓取点/放置点示教；
- lift_robot: 完整竞赛流程；
